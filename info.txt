Crear component Vue:

Dirigirse a la ruta del proyecto frontend/src/components:

Crear nuevo Vue Component.

Librerías de Vue que facilitan el desarrollo en typescript:

https://github.com/vuejs/vue-class-component
https://github.com/kaorun343/vue-property-decorator

Editar el fichero .vue que hemos creado en el proyecto.

Copiar en la parte de "script" la funcionalidad básica de un componente Vue para typescript.

<script lang="ts">
  import {Vue, Component} from 'vue-property-decorator';

  @Component({
    name: 'your-component-name',
  })
  export default class YourComponent extends Vue {
  }
</script>

Añadir el componente en nuestra vista (frontend/src/views/app-dashboard.vue).

Copiar el nombre que le hemos asignado al nuevo componente en la parte de la vista donde queramos visualizarlo.
